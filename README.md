## What is A.N.N.A.

A.N.N.A. is a federated autonomous agent system built on a weightless cognitive architecture. Intelligence emerges from accumulated tokenization pattern history rather than static weight matrices. The system operates entirely at the edge, requiring no persistent cloud dependency for core cognition. Every inference, every decision, every learning cycle runs within the user's sovereign compute perimeter.

The stack spans six Rust crates covering autonomous cognition, vision processing, molecular synthesis planning, federated learning, voice interaction, and zero-trust cryptographic session management. A React frontend delivers the user-facing interface across web and mobile targets.

## Architecture

```
A.N.N.A. Stack
├── anna-core         Cooper Tristate Theorem decision loop
├── anna-vision       Edge vision processing, EXIF anonymization, Gemini multimodal
├── anna-memory       SurrealDB palate state, SHA-512 audit chain
├── anna-federated    Wasmtime isolated learning, gradient-only uplink
├── anna-api          Axum REST layer, Firebase sync
├── anna-crypto       i-Plane winding tokens, zero-ID session authentication
├── anna-synth        Bio Quantum Assembly Combinator, ZZFeatureMap, Navier-Stokes
├── anna-learning     Pattern recognition, federated gradient computation
└── anna-voice        Piper TTS, Vosk STT, Sherpa-onnx STT, MoodAnalyzer
```

**Frontend:** React + TypeScript + Vite  
**Backend:** Rust workspace (Axum, SurrealDB, Wasmtime)  
**AI Integration:** Vertex AI Gemini multimodal  
**Realtime:** Firebase Realtime Database  
**Local Voice:** Piper TTS, Vosk STT, Sherpa-onnx  
**Cryptography:** SHA-512 Cooper Protocol sealing, i-Plane winding accumulator
